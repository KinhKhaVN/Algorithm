## BENCHMARK THE RUNNING TIME OF SEARCHING ALGORITHMS

- [x] Linear search : $O(n)$
- [x] Binary seach  : $O(n)$

## BENCHMARK THE RUNNING TIME OF SORTING ALGORITHMS

- [x] BubbleSort   : $O(n^2)$
- [x] SelectionSort: $O(n^2)$
- [x] InsertionSort: $O(n^2)$
- [x] QuickSort
    - Best: $O(nlog(n))$
    - Worsrt: $O(n^2)$
- [x] MergeSort    : $O(nlog(n))$
- [x] HeapSort     : $O(nlog(n))$
- [x] ShakerSort
    - Best: $O(nlogn)$
    - Worsrt: $O(n^2)$
